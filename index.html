ตั้งค่า Firebase

วางค่า Firebase config ของโปรเจกต์คุณด้านล่าง ระบบจะเก็บไว้ในเบราว์เซอร์เครื่องนี้

ก่อนใช้ ต้องเปิดใน Firebase

  1. Authentication → เปิด Email/Password
  2. Firestore Database → Create database
  3. Rules เบื้องต้น: ให้ user อ่าน/เขียนได้เฉพาะข้อมูลของตัวเอง
Firebase config จะไม่ถูกฝังตายตัวในไฟล์นี้ · เหมาะสำหรับนำไปอัป GitHub Pages แล้วตั้งค่าใช้งานจริง
