สำหรับ training ภายใน
1. ให้พนักงานกรอกชื่อก่อนใช้งาน 2. เปิดอ่านหัวข้อที่จำเป็นของ outlet ตัวเอง 3. ทำ quiz หลังจบการเทรนแต่ละหมวด 4. ใช้ notes บันทึก script การพูดกับแขก, wine pairing, allergy info และ service standard 5. ถ้าจะทำเป็นระบบออนไลน์ต่อ ให้ใช้หน้า Firebase Setup เป็นจุดเริ่มต้น

Firebase Setup (เตรียมต่อออนไลน์)

เวอร์ชันนี้ยังใช้งานได้แบบ local ก่อน

หากต้องการให้หลายเครื่องเห็น progress เดียวกัน ให้เชื่อม Firebase โดยเพิ่มไฟล์ config แล้วแก้จุดบันทึกข้อมูลจาก localStorage ไป Firestore/Realtime Database ภายหลังได้ทันที

ตัวอย่าง object ที่ต้องใส่:

{ apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT.firebaseapp.com", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT.appspot.com", messagingSenderId: "...", appId: "..." }
ข้อมูลที่ควรเก็บใน Firebase: - staffProfiles - lessonProgress - quizHistory - personalNotes - favorites หากต้องการ ฉันสามารถทำเวอร์ชันถัดไปที่เชื่อม Firebase จริงให้เลยได้
