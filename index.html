สำหรับ training ภายใน

1. ให้พนักงานกรอกชื่อก่อนใช้งาน เพื่อเก็บ progress แยกตามเครื่อง

2. เปิดอ่านเป็นชุดการเรียน เช่น English for F&B หรือ Allergy & Hygiene ตาม outlet ที่ต้องใช้

3. ใช้ Reader Mode เปิดสอนบนจอหรือ iPad ได้ เพราะเนื้อหาจะขึ้นเต็มหน้าแล้ว

4. หลังจบแต่ละชุด ให้ทำ Quiz และให้หัวหน้างานทบทวนจาก Notes

5. ภายหลังสามารถเชื่อม Firebase เพื่อให้หลายเครื่องเห็น progress เดียวกันได้

Firebase Setup (โครงเตรียมไว้)

สำหรับทำเวอร์ชันหลายเครื่องในอนาคต

ตอนนี้เวอร์ชันนี้ยังเก็บข้อมูลใน LocalStorage ของเครื่องเดิมก่อน เพื่อให้ใช้งานได้ทันทีแบบ offline

ถ้าจะเชื่อม Firebase ในเวอร์ชันถัดไป ให้เพิ่มข้อมูล config แล้วเปลี่ยนส่วน persist / load ให้ไปอ่านจาก Firestore แทน

const firebaseConfig = { apiKey: "YOUR_API_KEY", authDomain: "YOUR_PROJECT.firebaseapp.com", projectId: "YOUR_PROJECT_ID", storageBucket: "YOUR_PROJECT.appspot.com", messagingSenderId: "YOUR_SENDER_ID", appId: "YOUR_APP_ID" };
ถ้าหน่อยต้องการ เวอร์ชันถัดไปฉันสามารถทำต่อให้เป็น login จริง + sync progress + quiz score ข้ามหลายเครื่องได้
