Laya F&B Knowledge Hub
======================

ไฟล์นี้เป็นเว็บแอพสำหรับเปิดอ่านความรู้ด้าน Food & Beverage

วิธีใช้งาน
1) แตกไฟล์ zip
2) เปิดไฟล์ index.html ได้ทันที
หรือ
3) อัปโหลดทั้งโฟลเดอร์ขึ้น GitHub Pages / Netlify / โฮสต์เว็บอื่น ๆ

ฟังก์ชันที่มี
- ค้นหาหัวข้อ
- เลือกหมวดหมู่
- กดดาวเก็บเป็นรายการโปรด
- เขียนโน้ตส่วนตัว (บันทึกในเครื่องผ่าน Local Storage)
- มีบทความตัวอย่างด้าน F&B หลายหมวด

ไฟล์ที่สามารถแก้เพิ่มเนื้อหาได้
- data.js  = เพิ่ม/ลบ/แก้บทความความรู้
- styles.css = ปรับสีและหน้าตา
- app.js = ปรับการทำงาน

ถ้าต้องการเวอร์ชันต่อไป สามารถเพิ่มได้ เช่น
- ระบบ login
- quiz ทดสอบพนักงาน
- แยกหมวด Wine / Coffee / Service ลึกขึ้น
- เชื่อม Firebase เพื่อเก็บโน้ตหรือบทเรียนออนไลน์
- เพิ่มรูปภาพหรือ PDF แนบในแต่ละหัวข้อ
